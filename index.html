Netlify Proxy App

This app proxies requests to a configurable target URL.

Proxy endpoint: /.netlify/functions/proxy

By default, it returns your IP address. Configure environment variables in Netlify dashboard to change the target URL and add proxy settings.

Testing

Click the link above or visit https://boisterous-kelpie-5172c1.netlify.app/.netlify/functions/proxy in your browser.

If proxy is configured, it will show your proxy's IP instead of Netlify's.
